# w3id.org/taktology

Permanent identifier for **taktology** — a minimal, research-grounded RDF
vocabulary for takt production planning in construction (takt zones, wagons,
takt tasks, crews and the takt-plan graph), aligned to the Digital Twin
Construction (DTC) ontology, BOT and the TopologicPy ontology.

- **Namespace:** `https://w3id.org/taktology#` (hash namespace; preferred
  prefix `takt`)
- **Source repository:** <https://github.com/byggstyrning/taktology>
- **License:** CC BY 4.0

## Redirects

| Request | Target |
|---|---|
| `Accept: text/turtle` (or `application/x-turtle`) | `ontology/takt.ttl` on the repo's `main` branch |
| `/taktology/X.Y.Z` (version IRIs, e.g. `owl:versionIRI`) | `ontology/takt.ttl` at git tag `vX.Y.Z` |
| `/taktology/context` | `contexts/takt.context.jsonld` (JSON-LD context) |
| anything else (browsers) | the repository documentation page |
